Saltelli Estimators of Global Sensitivity

Applies to a model $Y = f(x_1, \ldots, x_d)$.

  1. Construct two matrices of samples: $A_{ij} = x^{(i)}_j$ for $i = 1 \ldots M$ and $j = 1 \ldots d$, and $B_{ij}$ a copy of the same with i.i.d. entries.
  2. Construct matrices $C_1, \ldots, C_d$ where $[C_i]_{kj} = B_{kj}$ if $k \not = i$ and $[C_i]_{kj} = A_{kj}$ if $k = i$.
  3. Run the model on each of the $(d+2)M$ input sets, letting $Y_A = f(A), Y_B = f(B), Y_{C_i}=f(C_i)$.
  4. Set $f_0 = \E[Y]$ and estimate sensitivities

$$ \hat{S}_i = \frac{\frac{1}{M} Y_A^T Y_{C_i} - f_0^2}{\frac{1}{M} Y_A^T Y_A - f_0^2} $$

This is essentially an estimator of $$ \begin{aligned} S_i & = \frac{\Cov_{x_i}(f(x_i, x_{-i}), f(\tilde{x}_i, x_{-i}))}{\Var(f(x))} \stackrel{?}{=} \frac{\Var(\E[Y \mid X_i])}{\Var(y)} \\ S_{T_i} & = 1 - \frac{\Var(\E[Y \mid X_{-i}])}{\Var(Y)} \end{aligned} $$
