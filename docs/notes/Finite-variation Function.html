Finite-variation Function

A function $f : [0, T] \to \R$ is has finite variation if it holds that $$ \sup \sum_{i=1}^p |f(t_{i}) - f(t_{i-1})| < \infty $$ where the supremum is taken over all integers $p \geq 1$ and increasing sequences $\{t_i\}_{i=1}^p \subseteq [0, T]$. In fact, for any sequence subdivisions $0 < t^{n}_0 < t^{n}_1 < \ldots < t^n_{p_n}=T$ for which the mesh size $\sup_{i=1\ldots p_n} |t_{i} - t_{i-1}|$ converges to zero, the supremum is attained by $$ \lim_{n \to \infty} \sum_{i=1}^{p_n} |f(t^n_i) - f(t^n_{i-1})| $$ and this quantity is called the variation of $f$ on $[0, T]$.

Measure-theoretic Representation

Definition (measure-theoretic perspective on finite variation). we say that $f : [0, T] \to \R$ has finite variation if there exists a finite signed measure $\mu$ such that $f(t) = \mu([0, t])$ for each $t \in [0, T]$. It is common to assume also that $f(0)=0$ and that $f$ is continuous.

In light of these, the previous definition based on partitions can be viewed as taking a simple function approximation to $1_{[0, T]}(x)$, since $$ \lim_{n \to \infty} \sum_{i=1}^{p_n}|f(t^n_i)-f(^n_{i-1})| =\mu([0, T]) < \infty. $$ The measure theoretic perspective can also be used to define a notion of integration with respect to finite variation processes. First, we make some definitions.

Definition (total variation measure). suppose that $f\in C^0([0, T])$ has finite variation, that $f(0)=0$, and let $\mu$ such that $f(t) = \mu([0, t])$. Let $\mu^+, \mu^-$ be the Jordan decomposition of $\mu$, ie. a unique decomposition $\mu = \mu^+ - \mu^-$ where $\mu^+$, $\mu^-$ are positive measures with disjoint support.

The measure $|\mu| \coloneqq \mu^+ + \mu^-$ is called the total variation measure of $A$.

Remarks:

  1. The total variation measure induces the following notions of integration with respect to $df(t)$. $$

\int_0^T g(s) \, df(s) \coloneqq \int_0^T g(s) \ \mu(ds) \qquad \int_0^T g(s) \, |df(s)| \coloneqq \int_0^T g(s) \, |\mu|(ds) $$

  1. The measures $\mu^+$, $\mu^-$ define a decomposition of $f(t) = f^+(t) - f^-(t)$ given by $f^+(t) =\mu^+([0, t])$ and $f^-(t) = \mu^-([0, t])$, which are both positive valued, finite, monotonically increasing functions. Conversely, any function that is the difference of two positive finite monotonically increasing functions is finite variation.

For a given continuous function $g(t)$, integration with respect to $f(t)$ yields a new function $G(t) \coloneqq \int_0^t g(s) \, df(s)$. It is natural to ask whether $G(t)$ is also finite variation.

Proposition (closure of F.V. functions wrt. integration) for continuous $g(t) : [0, T] \to \R$ and for finite variation $f(t) : [0, T] \to \R$, the function $$ G(t) = \int_0^t g(s) \, df(s) $$ is finite variation if $$ \int_0^T |g(s)| \, |df(s)| < \infty. $$

Proof sketch. similar to simple function approximations of the Lebesgue integral, we can represent $G(t)$ using a sequence $(P^n)_{n \geq 1}$ of partitions $0 = t^n_0 < t^n_1 < \ldots < t^n_{p_n}=T$, for which it holds $$ G(T) = \lim_{n \to \infty} \sum_{i=1}^{p_n}g(t_i)|f(t_i) - f(t_{i-1})| $$ Then, the condition for $G$ to have finite variation is that $$ \infty > \lim_{n \to \infty} \sum_{i=1}^{p_n} |g(t_i)| |f(t_i) - f(t_{i-1})| $$ and the right hand side is none other than $\int_0^T |g(s)| \, |df(s)|$.
