Hypothesis Testing

Simple Hypothesis Testing

In a simple hypothesis test, we observe a random variable $Y \sim P_0$, and we would like to determine whether its distribution is $P = P_0$ (the 'null' hypothesis) or it is $P = P_1$ (the 'alternative' hypothesis). In Hypothesis Testing, we aim to write a statistic $\Psi(Y) \in \{0, 1\}$ which 'distinguishes' the distributions, in the sense that its distribution under $P_0$ is as-different-as-possible as the distribution under $P_1$.

The key metrics are:

  • Type I Error: $P_0(\Psi(Y) = 1)$, known as a false positive or _false alarm.
  • Type II Error: $P_1(\Psi(Y) = 0)$, known as a false negative or missed detection.
  • Power: $P_1(\Psi(y) = 1)$, known as a true positive.

These metrics often need to be 'scalarized,' or converted to a single number. Some options include: Weighted Probability of Error: minimize a weighted sum of T1 + T2 errors: $\Ecl_\omega = (1-\omega) P_0[\Psi(Y) \not = 0] + \omega P_1[\Psi(y) \not = 1]$. Neyman-Pearson Criterion: maximize power subject to constraint on T1 error: $\Ecl^{NP}_\alpha = P_1(\Psi(Y)=1)-1\{P_0[\Psi(Y) \not = 0] \leq \alpha\}$ where the indicator is infinite if the constraint does not hold.

Composite Hypothesis Testing

Typically, a hypothesis takes the form $H_i : y \sim P_i$, and when the hypothesis holds the samples are drawn from a specific distribution. In a composite test, the hypothesis may only specify a set of possible distributions $y \sim P_\theta \in \Pcl$ where $\Pcl = \{P_\theta : \theta \in \Theta\}$ for some parameters $\Theta$.

Note: Composite Hypothesis Testing is often solved with a Generalized Likelihood Ratio Test. The GLRT is not to be confused with a Composite Likelihood Ratio Test (Lindsay, 1988) which tests based on a sum of log-likelihoods.

Some scalarized error metrics in composite testing are:

  • Bayesian Error Probability: we assume there is a known prior $\pi$ on $\Theta$ and that, under the alternative, the data distribution is drawn at random from the prior: $P_1(Y \in \Acl) = \E_{\theta \sim \pi} P_\theta(Y \in \Acl)$. In this setting, composite testing is reduced to simple testing $H_0 : P_0$ against $H_1 : Q_\pi$, where $Q_\pi(Y \in \Acl) = \E_{\theta \in \pi}[P_\theta(Y \in \Acl)]$ is the posterior.
  • Minimax Error Probability: in which the goal is to minimize (or bound),

$$ \Ecl_{\mathsf{mm}}(P_0, \Pcl, \Psi) = \sup_{\theta \in \Theta} \Ecl_{1/2}(P_0, P_\theta, \Psi). $$ In some cases, this definition of minimax testing error is trivial, since (for example) if $P_0 \in \Pcl$ then it is impossible to achieve better than random guessing $\Ecl_{\mathsf{mm}} = 1/2$. More generally, if there exists a sequence $P_{\theta_n}\to P_0$ where $\{\theta_n\}_{n \geq 1} \subseteq \Theta$, then the hypothesis test can be made arbitrary difficult. This motivates the following scalarization.

  • Minimax Testing Radius: suppose that there exists $\theta_0 \in \Theta$ such that $P_0 = P_{\theta_0} \in \Pcl$. The minimax testing radius is the smallest $\eps > 0$ such that the set $\Pcl_\eps = \{p_\theta : \theta \in \Theta \setminus B_\eps(\theta)\}$ has testing error,

$$ \Ecl_{\mathsf{mm}}(P_0, \Pcl_\eps, \Psi) \leq 1/2 - \delta $$ where $\delta > 0$ is an arbitrary constant.
