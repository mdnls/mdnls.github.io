Gaussian Process

A Gaussian Process is a stochastic process $X$ for which any projection $(X(x_1), \ldots, X(x_d)), \{x_1,\ldots, x_d\} \in \Dcl$ follows a multivariate Gaussian distribution. Equivalently, if all (finite) linear combinations $R \in \mathsf{span} \{X(s)\}_{s \in \Dcl}$ follow a univariate Gaussian distribution.

A GP is entirely determined by its mean function and covariance kernel.

  • Mean Function: $\mu(s) = \E[X(s)]$
  • Covariance Kernel: $C(s, s') = \Cov[X(s), X(s')]$

Some important kinds of GPs are:

  • Stationary Gaussian Process: has the property that $\mu(s) = \mu$ and $C(s, s') = C(s - s')$. This implies that $\Var(X(s)) = C(s-s) = C(0)$ for all $s \in \Dcl$. Furthmore, if $C = C(\|s-s'\|)$ where $\|\cdot\|$ is a norm on $\Dcl$, then the
  • Isotropic Stationary GP: $C(s, s') = C(\|s-s'\|)$ where $\|\cdot\|$ is a norm on $\Dcl$.

Bochner's Theorem

For a discrete index set $T \subseteq [n]$, one may think of a stationary covariance kernel as a circulant matrix: $C(i, j) = c(i-j)$ for some $c : \Z \to \R$. In this case, the covariance matrix is diagonalized by the discrete Fourier transform. Bochner's Theorem gives a generalization to continuous index sets.

Informal Statement (cf. LeGall Theorem 1.11). A stationary GP $(X_t)_{t \in \R}$ has covariance kernel that may be written in the form $$ \Gamma(s, t) = \int e^{i \zeta(t-s)} \, \mu(d \zeta) $$ for some measure $\mu$ on $\R$. This measure is known as the spectral measure of the process.

Examples of Covariance Kernels

A GP can be a good choice of model for a random field. In this case, the covariance kernel controls properties of the variance of the field, and adjusting the kernel induces 'roughness' or 'smoothness' in sampled fields. Some popular examples are:

  • Exponential Kernel: $C(t_1, t_2) = \tau^2 \exp \left( - \lambda^{-1}|t_1-t_2|\right)$.
  • Gaussian Kernel: $C(t_1, t_2) = \tau^2 \exp \left(-2\lambda^2 |t_1-t_2|^2\right)$.
