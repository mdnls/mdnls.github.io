Stochastic Processes

Let $\Pcl = (\Omega, \Fcl, P)$ be a generic probability space. A stochastic process $(X_t)_{t \in I}$ is a collection of random variables, indexed by some set $I$, which are all measurable with respect to $\Pcl$. Stochastic processes are extremely important objects in probability and computer science. See also:

  • Brownian Motion
  • Empirical Process Theory

Continuity and Differentiability

For any $\omega \in \Omega$, the mapping $t \mapsto X_t(\omega)$ is a sample path of the stochastic process. We often think of processes like Brownian Motion, which has continuous sample paths. There are in fact multiple ways in which a stochastic process can be continuous:

  • Continuity in mean: $\mu(t) \coloneqq \E[X_t]$ is continuous.
  • Continuity in probability: $t \to t_0$ implies $X(t) \stackrel{p}{\longrightarrow} X(t_0)$.
  • Continuity in $\Lcl^2$: $t \to t_0$ implies $\|X(t) - X(t_0)\|_{\Lcl^2(\Omega)} \to 0$.
  • Continuity of sample paths: for almost every $\omega \in \Omega$, $X(\cdot, \omega)$ is continuous.

Generally speaking, if the random variables converge $X_t \to X_{t_0}$ whenever $t \to t_0$, in some mode of probabilistic convergence, then the process is said to be continuous in that mode of convergence.

Similar to the notion of continuity, the differentiability of a random process is described by the convergence of random variables of the form, $$ X_{t + \Delta} \coloneqq \frac{X_{t+\Delta} - X_t}{\Delta} $$ as $\Delta \to 0$. For example, to say that a process is differentiable in $\Lcl^2$ implies that as $\Delta \to 0$ the r.v. $X_{t+\Delta}$ converges in $\Lcl^2(\Omega)$. By completeness of $\Lcl^2$, there exists a limiting random variable $X'_t = \lim_{\Delta \to 0} X_{t + \Delta}$.

Concentration Properties

It is often useful to make probabilistic assumptions about the process increments $X_t - X_s$. The assumption that $\E[X_t - X_s \mid \Fcl_s] = 0$ describes the class of martingales, which automatically inherent properties like sample path regularity from this assumption.

Alternatively, one may wish to assume concentration properties of these increments.

Definition (sub-gaussian process). a process $(X_t)_{t \in T}$ is sub-Gaussian if $\E[X_t] = 0$ for all $t \in T$, and if $$ \log \E[e^{\lambda(X_t - X_s)}] \leq \frac{\lambda^2 d(t, s)^2}{2} $$ where $d(t, s)$ is a metric on $T$.

The sub-gaussianity assumption can be used to prove bounds on $\E[\sup_{t \in T} X_t]$ (cf. Empirical Process Theory, Dudley's Entropy Integral).

Time-dependent Processes

One common situation is when $I = [0, \infty]$, so that $t \in I$ can be interpreted as time and the collection $(X_t)_{t \geq 0}$ can be viewed as a random time-dependent function. To discuss measure-theoretic aspects of these random variables, it is often helpful to add some assumptions:

Definition (Adapted/Progressive/Measurable Processes). suppose that $\Fcl$ admits a filtration $(\Fcl_t)_{t \geq 0}$. Then:

  • $(X_t)_{t \geq 0}$ is said to be adapted if, for each $t \geq 0$, $\omega \mapsto X_t(\omega)$ is $\Fcl_t$-measurable.
  • $(X_t)_{t \geq 0}$ is said to be measurable if $(t, \omega) \mapsto X_t(\omega)$ is measurable with respect to $\Bcl(\R_+) \otimes \Fcl$.
  • $(X_t)_{t \geq 0}$ is said to be progressive if, for every $t \geq 0$, the mapping $(s, \omega) \mapsto X_s(\omega)$ is measurable with respect to $\Bcl([0, t]) \otimes \Fcl_t$.

Here, $\Bcl(I)$ is the Borel $\sigma$-algebra of the interval $I$.

Clearly, a progressive process is both measurable and adapted. An adapted process is not necessarily progressive, since this requires extra assumptions on the regularity of $X_t$ in time:

Proposition (right/left continuous adapted processes are progressive) suppose $(X_t)_{t \geq 0}$ is adapted and that $t \mapsto X_t(\omega)$ is right-continuous for $P$-a.e. $\omega \in \Omega$. Then it is also progressive.

If $t \mapsto X_t(\omega)$ is left-continuous, then a similar argument implies that it is progressive.

Proof. the basic idea is to make use of the fact that since $(X_t)_{t \geq 0}$ is right continuous, we can write it as the limit of discretizations $X^n_s(\omega) = X_{\lceil ns \rceil /n}(\omega)$ (also setting $X^n_s(\omega) = X_t(\omega)$ whenever $\lceil n s \rceil / s > t$). For fixed $n \geq 0$, adaptivity implies that $X^n_s(\omega)$ is progressive: for $A \in \Bcl(\R)$, $$ \begin{aligned} \{(\omega, s) \in \Omega \times [0, t] : X^n_s(\omega)\in A\} & = \{ X_t(\omega) \in A\}\times \{t\} \cup \left( \bigcup_{k =0}^{\lfloor n t \rfloor} \left\{ X_{k/n}(\omega) \in A\right\} \times \{s : (k-1)/n < s \leq k/n \}\right) \end{aligned} $$ which is clearly measurable. The proof follows from the fact that the limit $t\mapsto \lim_{n \to \infty}X^n_t(\omega)$ is therefore also measurable.

It is also possible to characterize progressive processes as those which are measurable with respect to a certain $\sigma$-algebra.

Proposition (progressive $\sigma$-algebra). consider the $\sigma$-algebra $\Pcl$ that is generated by the sets $A \in \Fcl \otimes \Bcl(\R_+)$ for which $X_t = 1_A(\omega, t)$ is a progressive process. In other words, if $A \in \Pcl$, then for each $t \geq 0$, $A \cap \Omega \otimes [0, t] \in \Fcl_t \otimes \Bcl([0, t])$.

Then, a process $Y_t$ is progressive iff the mapping $(\omega, t) \mapsto Y_t(\omega)$ is measurable with respect to $\Pcl$.

Proof.

  1. If $Y_t$ is progressive, then for any $A \in \Bcl(\Rcl)$, the set $Y^{-1}_t(A) = \{(\omega, s) \in \Omega \otimes [0, t]: Y_s(\omega) \in A\}$ belongs to $\Fcl_t \otimes \Bcl([0, t])$. This implies that $Y^{-1}(A) = \{(\omega, s) \in \Omega \otimes \R_+ : Y_s(\omega) \in A\}$ belongs to $\Pcl$.
  2. Similarly, if $Y^{-1}(A)$ is measurable with respect to $\Pcl$, then $Y_t^{-1}(A) = Y^{-1}(A) \cap (\Omega \otimes [0, t])$ is measurable with respect to $\Fcl_t \otimes \Bcl([0, t])$ for each $t\geq 0$ and so $Y_t$ is progressive.
