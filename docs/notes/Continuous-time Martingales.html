Continuous-time Martingales

A martingale is a type of stochastic process $(X_t)_{t \geq 0}$ whose increments $X_t - X_s$ behave in a predictable way. For a true martingale, the increments have mean zero, and for sub or supermartingales, the increments have positive and negative means (respectively). Such processes are useful in many areas of stochastic processes.

Definition (martingales, sub- and super- martingales). let $(\Omega, (\Fcl_t)_{t \geq 0}, P)$ be a filtered probability space and suppose that $(X_t)_{t \geq 0}$ is a random process that satisfies one of the following.

  1. Submartingale. for $t \geq s$ it holds $\E[X_t \mid \Fcl_s] \geq X_s$.
  2. Supermartingale. for $t \geq s$ it holds $\E[X_t \mid \Fcl_s] \leq X_s$.
  3. Martingale. for $t \geq s$, $\E[X_t \mid \Fcl_s] = X_s$.

Some important examples of martingales can be constructed from a generic process $(X_t)_{t \geq 0}$ that has independent increments (ie. $X_t - X_s$ is independent of $\Fcl_s$ for $t > s$). When this is the case, the following processes are martingales:

  1. Centered process: $Y_t \coloneqq X_t - \E[X_t]$.
  2. Variance process: $Z_t \coloneqq Y_t^2 - \E[Y_t^2]$.
  3. Exponential martingale: $M_t = e^{\theta X_t} / \E[e^{\theta X_t}]$.

Martingale Regularity

When constructing Brownian motion, one must demonstrate that it is possible to construct a random process with continuous sample paths whose marginals satisfy the right Gaussianity properties. One way is to start with a Gaussian White Noise and argue that it admits a modification with continuous sample paths. In a similar spirit, we can prove under quite general conditions that a supermartingale $(X_t)_{t \geq 0}$ admits a modification with right continuous sample paths.

Theorem (martingale regularity). Suppose that $(\Fcl_t)_{t \geq 0}$ is right continuous and complete. Let $(X_t)_{t \geq 0}$ be a supermartingale such that $t \mapsto \E[X_t]$ is right-continuous.

Then $(X_t)_{t \geq 0}$ admits a modification with c치dl치g sample paths, which is also an $\Fcl_t$ supermartingale.

Proof sketch. The full proof of this theorem can be found in LeGall's book:

  1. Similar to the construction of brownian motion, the general strategy is to prove that there exists a countable dense subset $D \subseteq \R$ for which the restriction $(X_t)_{t \in D}$ has the right regularity properties. In particular, it is true $P$-a.s. that the limit exists, $$

X_{t+}(\omega) \coloneqq \lim_{D \ \ni \ s \to t+} X_s(\omega) $$and we take $N$ to be the (null) set of $\omega \in \Omega$ where the limit does not exist. This fact can be proven with Doob's upcrossing inequality + an analytical lemma that basically says 'bounded upcrossing numbers implies existence of right limits.'

  1. The process $X_{t+}(\omega)$ is right-continuous by construction and it also happens to be measurable with respect to $\Fcl_{t+}$. From the supermartingale property, $X_t \geq \E[X_{t+} \mid \Fcl_t]$. By standard convergence theorems, we have $\E[X_{t+}] = \lim_{s \to t+} \E[X_s]$, so the assumption $t \mapsto \E[X_t]$ is right continuous implies that $\E[X_{t+}] = \E[X_t]$, which forces $X_t = \E[X_{t+} \mid \Fcl_t]$.
  2. Finally, if $\Fcl_t$ is right continuous, then for each $t \geq 0$, we have for a.e. $\omega \in \Omega$ (specifically $\omega \not \in N$) that $X_t = \E[X_{t+} \mid \Fcl_t] = X_{t+}$, which implies that $(X_{t+})_{t \geq 0}$ is a modification of $(X_t)_{t \geq 0}$.

In the end, setting $$ Y_t(\omega) = \begin{cases} X_{t+}(\omega) & \omega \not \in N \\ 0 & \text{else} \end{cases} $$ yields a process that is $\Fcl_t$-measurable (here we use completeness, since $N \in \Fcl_t$ for all $t \geq 0$), that is a modification of $(X_t)_{t \geq 0}$, and that is a c치dl치g supermartingale.

Martingale Boundedness

There are a few different ways to assume that a given martingale $(X_t)_{t \geq 0}$ is bounded, and in this section we give some definitions and relationships.

Definition (Martingale Boundedness). a martingale $(X_t)_{t \geq 0}$ can satisfy one of the following:

  1. $\Lcl^1$ boundedness: there exists $M < \infty$ with $\sup_{t \geq 0} \E[|X_t|] < M$.
  2. Uniform integrability: the process satisfies $\lim_{M \to \infty} \sup_{t \geq 0} \E[|X_t| \cdot 1 \{X_t \geq M\}] = 0$.
  3. $\Lcl^p$ boundedness: for $p > 1$, there exists $M < \infty$ with $\sup_{t \geq 0} \E[|X_t|^p] < M$.

In fact, $(3) \Rightarrow (2) \Rightarrow (1)$.

Proof. to see $(3) \Rightarrow (2)$, note that for $p > 1$ $$ \E[|X_i| \cdot 1\{X_i \geq M\}] \leq \E\left[ |X_i| \left( \frac{|X_i|^{p-1}}{M}\right)\right] \leq \frac{\sup_{t \geq 0} \E[|X_i|^p]}{M^{p-1}} \stackrel{M \to \infty}{\longrightarrow} 0. $$ To see $(2) \Rightarrow (1)$, use that $\E[|X_t|] \leq M + \E[|X_i| \cdot 1 \{ |X_t| \geq M\}] < M + \eps$ for $M = M(\eps)$ sufficiently large.

Uniform integrability can also be characterized using test functions, which is more convenient in some cases.

Proposition (test functions of uniform integrability) a function $\phi : \R \to \R$ is a test function of uniform integrability if $$ \lim_{x \to \infty} \frac{\phi(x)}{x} = \infty. $$ Given such a test function, $(X_t)_{t\geq 0}$ is uniformly integrable if and only if $\sup_{t \geq 0} \E[\phi(|X_t|)] < \infty$.

Proof. the general idea is that since $\phi(x)$ grows faster than any linear function, the expectation can only be finite if $X_t$ decays sub-linearly. If $\sup_{t \geq 0} \E[\phi(|X_t|)|] = C$, then by the growth condition, for any $n \geq 1$ there exists $k(n)$ so $\phi(x) \geq n C x$ whenever $x \geq k$. Therefore $$ C \geq \sup_{t \geq 0} \E[\phi(|X_t|)] \geq \sup_{t \geq 0} \E[n C |X_t| \cdot 1 \{X_t \geq k(n)\}] \implies \sup_{t \geq 0} \E[|X_t| \cdot 1 \{ X_t \geq k(n)\}] \leq \frac{1}{n} $$ and the proof follows because $n$ is arbitrary. To prove that this condition is necessary, construct a function $g : \R_+ \to \R_+$ for which $\int_0^\infty g(x) \, dx = \infty$ but $\int_0^\infty g(x) f(x) \, dx<\infty$ for any $f$ that decays at infinity.

Martingale Convergence

It is often valuable to know whether a given martingale (or supermartingale) converges to a limiting random variable as $t \to \infty$.

Proposition (almost sure limit of bounded supermartingales). suppose that $(X_t)_{t \geq 0}$ is a supermartingale with right-continuous sample paths. Suppose further that the process is $\Lcl^1$-bounded in the sense that $\sup_{t \geq 0}\E[|X_t|] < M < \infty$. Then, there exists $X_\infty \in \Lcl^1$ such that $X_t \stackrel{a.s.}{\longrightarrow} X_\infty$.

Proof. basically, the idea is that if $(X_t)_{t \geq 0}$ is uniformly bounded, then Doob's upcrossing inequality reads $$ \E[M^X_{ab}(\Q)] \leq \sup_{t \geq 0} \E[(X_t - a)^-] < \infty $$ and so the standard martingale regularity argument implies that $X_\infty \coloneqq \lim_{Q \ \ni \ s \to \infty} X_s$ exists almost surely. Moreover by Fatou, $\E[|X_\infty|] \leq \lim \inf_{t \to \infty} \E[|X_t|] \leq \infty$ so $X_\infty$ is also $\Lcl^1$.

Propostion ($\Lcl^1$ limit of UI martingale). suppose that $(X_t)_{t \geq 0}$ is a martingale with right-continuous sample paths. Then the following are equivalent:

  1. Closedness. there exists $Z \in \Lcl^1$ for which $X_t = \E[Z\mid \Fcl_t]$ for each $t \geq 0$.
  2. Uniform integrability. the collection $(X_t)_{t \geq 0}$ is uniformly integrable.
  3. Convergence. $(X_t)_{t \geq 0}$ converges a.s. and in $\Lcl^1$ to a limit random variable $X_\infty \in \Lcl^1$.

Proof. $(1 \Rightarrow 2)$. let $\phi(x)$ be a convex test function of uniform integrability, then $\sup_{t \geq 0} \E[\phi(X_t)] \leq \E[\phi(Z)] < \infty$. The fact that the latter expectation is finite follows from the fact that $\E[\phi(Z)] < \infty$ is necessary for integrability of $Z$ (since this is the same as uniform integrability of $\{Z\}$).

$(2 \Rightarrow 3)$. if $(2)$ holds then already there exists a limit $X_\infty$ to which $X_t$ converges almost surely. UI implies uniform boundedness $\sup_{t \geq 0} \E[|X_t|] \leq M$ so by bounded convergence, $\lim_{t \to \infty}\E[|X_t - X_\infty|] = 0$.

$(3 \Rightarrow 2)$. under $(3)$, we aim to show that $X_t = \E[X_\infty \mid \Fcl_t]$ by proving that $X_t = \E[X_s \mid \Fcl_t] \to \E[X_\infty \mid \Fcl_t]$ as $s \to \infty$. This follows from Jensen's inequality: $$ \|\E[X_s\mid \Fcl_t] - \E[X_\infty \mid \Fcl_t]\|_1 = \E[|\E[X_s - X_\infty \mid \Fcl_t]|] \leq \|X_s - X_\infty\|_1 \stackrel{s \to \infty}{\longrightarrow} 0. $$
