Stopping Time

A stopping time is a random variable that represents the first occurrence of some property of a stochastic process.

Definition (Stopping Time). Let $(X_t)_{t \geq 0}$ be a real-valued stochastic process with respect to the filtration $(\Fcl_t)_{t \geq 0}$. A random variable $T \in [0, \infty]$ is a stopping time if $\{T \leq t\} \in \Fcl_t$ for each $t > 0$.

Additionally, if $T$ is a stopping time, we may define a $\sigma$-algebra $\Fcl_T$ as, $$ \Fcl_T = \{A \in \Fcl_{\infty} : \forall t \geq 0, \ A \cap \{T \leq t\} \in \Fcl_t\}. $$

Roughly speaking, $\Fcl_T$ represents the events whose presence can be determined from all the past information up to time $T$. As an example, if $T = \inf\{t \geq 0 : X_t \geq a\}$, then $1\{\sup_{t \geq 0} X_t \geq a\} = 1\{T < \infty\}$ is $\Fcl_T$ measurable.

Proposition. Let $(B_t)_{t \geq 0}$ be a Brownian Motion and let $T$ be a stopping time. Then the random variable $B_T$ defined by $$ B_T(\omega) = \begin{cases} B_{T(\omega)}(\omega) & T(\omega) < \infty \\ 0 & \mathsf{else} \end{cases} $$ is an $\Fcl_T$-measurable random variable.

Proof sketch. it's a bit strange to think of the index of $B_T$ as a random variable, so one proof strategy is to decouple the two. One way is to use the fact that for any $\eps > 0$, $B_{\lfloor nT \rfloor/n}$ is $\Fcl_T$ measurable since $\lfloor n T \rfloor / n \leq T$, and by continuity $B_T$ is still equal to the limit of $B_{\lfloor nT \rfloor / n}$ as $n \to \infty$.

Proof. to do the decoupling, we round $T$ to a strictly smaller value within the grid $\{k 2^{-n} : k \in \N\}$ and take $n \to \infty$.

  1. By continuity of brownian motion, it's true almost surely that

$$ B_T = \lim_{n \to \infty} \sum_{k\geq0} 1\{k 2^{-n} \leq T < (k+1)2^{-n}\} B_{k2^{-n}}.$$

  1. By definition $1\{T < (k+1)2^{-n}\}$ is $\Fcl_T$ measurable. We claim $B_k \coloneqq 1\{k2^{-n} \leq T\} B_{k2^{-n}}$ is also $\Fcl_T$ measurable. To check, we just need to show $\{B_k < t\} \in \Fcl_T$ for every $t \in \R$ (in other words, $B_k^{-1}((-\infty, t]) \in \Fcl_T$ for every $t \in \R$, so we're just checking that $B_k^{-1}$ maps measurable sets to measurable sets)
  2. We verify that for each $s \in [0 ,\infty)$, if $s < k2^{-n}$ then $\{B_k < t\}\cap \{T \leq s\} = \emptyset \in \Fcl_s$ whereas if $s \geq k2^{-n}$,

$$ \begin{aligned} \{B_k < t\} \cap \{T \leq s\} & = \underbrace{\{B_{k2^{-n}} < t\}}_{\in \Fcl_{k2^{-n}}\subseteq \Fcl_s} \cap \underbrace{\{k 2^{-n} \leq T \leq s\}}_{\in \Fcl_s} \end{aligned} $$

Stopping Times and Instantaneous Randomness

Proposition (stopping times & instantaneous randomness). let $T : \Omega \to [0, \infty]$ be a random variable. The following are equivalent:

  1. $T$ is a stopping time of $\Fcl_{t+}$.
  2. $\{T < t\} \in \Fcl_t$ for all $t >0$.
  3. The random variable $T \land t$ is $\Fcl_t$-measurable for each $t > 0$.

Furthermore, when $T$ is a stopping time of $\Fcl_{t+}$, then the corresponding stopped filtration $\Fcl_{T+}$ can be defined in two equivalent ways: $$ \begin{aligned} \Fcl_{T+} & = \{A \in \Fcl_\infty : \forall t > 0, A \cap \{T \leq t\} \in \Fcl_{t+}\} \\ &= \{A \in \Fcl_\infty : \forall t > 0, A \cap \{T < t\} \in \Fcl_t\} \end{aligned} $$

Generally speaking, the interesting content of this proposition are statements of the flavor that '$\{T < t\} \in \Fcl_t$ is equivalent to $\{T \leq t\} \in \Fcl_{t+}$'. If $(\Fcl_t)_{t \geq 0}$ is not right continuous then we may run into situations where $\{T = t\} \not \in \Fcl_t$ but $\{T = t\} \in \Fcl_{t+}$, so $T$ cannot technically be defined as a stopping time with respect to $\Fcl_t$. Instead, the above proposition describes how to treat $T$ as a stopping time by using $(\Fcl_{t+})_{t \geq 0}$.

Proof sketch.

  • (2 $\Rightarrow$ 1) we can write $\{T \leq t\} = \cap_{n \geq 1} \{T < t + 1/n\}$ which is measurable wrt $\Fcl_{t + 1/n}$ for any $n \geq 1$. As a consequence, $\{T \leq t\} \in \bigcap_{n \geq 1} \Fcl_{t + 1/n} = \Fcl_{t+}$.
  • (3 $\Rightarrow 2$) since the event $\{T < t\} = \{T \land t < t\} \in \Fcl_t$ for any $t > 0$.
  • (1 $\Rightarrow$ 3) by assumption, $\{T \leq t\} \in \Fcl_{q}$ for any $q > t$. In particular, for any $s < t$, we have $\{T \land t < s\} = \cup_{n \geq 1} \{T \leq s - 1/n\}$. By assumption, $\{T \leq s - 1/n\} \in \Fcl_{(s - 1/n)+} \subseteq \Fcl_t$, so the event $\{T \land t < s\}$ is a countable union of $\Fcl_t$-measurable events. If instead $s \geq t$, then $\{T \land t < s\} = \Omega$, so overall we have shown that $\{T \land t < s\} \in \Fcl_t$ for any $s \geq 0$.

The representation of $\Fcl_{T+}$ follows by similar arguments.

Stopping Times and Limits

Proposition (stopping times & limits) suppose $(S_n)_{n \geq 1}$ is a sequence of stopping times.

  1. If $S_n$ is monotone increasing, then $S = \lim_{n \to \infty} S_n$ is also a stopping time.
  2. If $S_n$ is monotone decreasing, then $S = \lim_{n \to \infty} S_n$ is a stopping time of $\Fcl_{t+}$ and $\Fcl_{S+} = \cap_{n \geq 1} \Fcl_{S_n+}$.
  3. If $S_n$ is monotone decreasing and stationary, in the sense that for each $\omega$ there exists $N(\omega) \in \N$ so that $S_n (\omega) = S(\omega)$ for all $n \geq N(\omega)$, then $S$ is also a stopping time and $\Fcl_{S} = \cap_{n \geq 1} \Fcl_S$.

Proof sketch. all these claims are proven by constructing $\{S \leq t\}$ or $\{S < t\}$ with countable unions/intersections of measurable sets. The intuition is that as $n \to \infty$, if the $S_n$ are monotone increasing then they behave like a 'left limit' in time, where as if they're monotone decreasing they behave as a 'right limit' in time.

Examples

Discretizing stopping times

Proposition (discretizing stopping times). suppose that $T$ is a stopping time and $S$ is an $\Fcl_T$ measurable r.v. with $S \geq T$ almost surely. Then $S$ is also a stopping time. In particular, setting $$ T_n(\omega) = \begin{cases} \frac{k+1}{2^n} & k 2^{-n} \leq T(\omega) \leq (k+1)2^{-n} \\ \infty & T(\omega) = \infty \\ \end{cases} $$ the r.v.s $T_n$ form a sequence of monotonically decreasing stopping times that approach $T$.

Entry Times

Proposition (entry times). let $(X_t)_{t \geq 0}$ be a adapted stochastic process taking values in a metric space $(E, d)$ and let $O$, $F$ be open and closed subsets of $E$ respectively. Then:

  1. If $(X_t)_{t \geq 0}$ has right continuous sample paths, then $T_O \coloneqq \inf \{t \geq 0 : X_t \in O\}$ is a stopping time of $\Fcl_{t+}$.
  2. If $(X_t)_{t \geq 0}$ has continuous sample paths, then $T_F \coloneqq \inf \{t \geq 0 : X_t \in F\}$ is a stopping time of $\Fcl_t$.
