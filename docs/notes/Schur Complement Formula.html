Schur Complement Formula

Definition: Let $X$ be a symmetric block matrix given by, $$ X = \begin{bmatrix} A & B \\ B^T & C \end{bmatrix} $$ where $A \in \R^{p \times p}$, $B \in \R^{p \times d}$, $C \in \R^{d \times d}$ where $A, C$ are invertible. The Schur complement relates the diagonal blocks of $X$:

  • $X / A \coloneqq C - B^T A^{-1} B$.
  • $X / C \coloneqq A - BC^{-1} B^T$.

Intuition: the Schur complement is a multivariate generalization of the 2x2 Matrix Determinant. It is useful because it allows to relate properties about the spectrum of $X$ to properties about the spectrum of its diagonal sub-matrices.

Proposition (Schur Complement Formula): $X \succcurlyeq 0$ iff $A \succcurlyeq 0$ and $X/A \succcurlyeq 0$. Proof. Consider the matrix $$\begin{aligned} G \coloneqq \begin{bmatrix} I & -B A^{-1} \\ 0 & I \end{bmatrix} \end{aligned}$$ for which $$ G^T X = \begin{bmatrix} A & B \\ 0 & C - B^T A^{-1} B\end{bmatrix}. $$ This matrix represents one step of block Gaussian elimination, where we have replaced $\mathsf{R2}$ by $\mathsf{R2} - B^T A^{-1} \mathsf{R1}$. So, if $X$ is PSD then any vector $v \in \mathsf{span}(e_1, \ldots, e_p)$ has $v^T X v \geq 0$. This implies $A$ is PSD, and the same argument works for $X/A$.

Further, $G^T X G = \diag(A, X/A)$. Clearly, $\det(G) = 1$ so that $G$ is invertible. It follows that if $A, X/A$ are PSD then any $v \in \R^{p + d}$ can be written $v = (G^{-1}s, G^{-1}t)$ and $v^T X v = s^T A s + t^T (X/A) t \geq 0$.

Proposition (Schur Determinant Formula): $\det(X) = \det(A) \det (X/A)$. Proof. this can be read off from the above display.
