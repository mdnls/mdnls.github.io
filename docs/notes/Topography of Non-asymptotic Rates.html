Description: a collection of notes related to important problems and/or instances of non-asymptotic statistical rates.

Nonparametric Regression

Estimate $\hat{f}(x)$ given $n$ i.i.d. samples $(x_i, y_i)$ of the form $y_i = f(x_i) + w_i$. with $x \sim P$.

  • Smoothing Estimators
    • With $f \in \mathsf{HÃ¶lder}(s, B)$ and $x \in \R$, the estimation error decays as $\sup_x \E[|\hat{f}(x) - f(x)|^2] \lesssim n^{-\frac{2s}{2s+1}}$.
    • With $\|f\|_{\mathsf{Lip}} \leq L$ and $x \in \R^d$, the estimation error decays as $\sup_x \E[|\hat{f}(x) - f(x)|^2] \leq n^{-\frac{2}{2+d}}$.
  • Orthogonal Series Estimators
    • With $f \in \mathsf{Sobolev}(s, B)$ and $x \in \R^d$, mean squared error decays as $\E_{x_1, \ ldots, x_n} [\|f - \hat{f}\|_{\Lcl^2(P)}^2] \lesssim n^{-\frac{2s}{2s+d}}$.
  • These results also apply nearly in-place for density estimation. In the above cases, setting $y_i = 1$ for each $x_i$, the smoothing methods become histogramming and orthogonal series estimators become moment estimators.

Gaussian Sequence Model

Multiple Testing Gaussian Sequence Model: multiple hypothesis testing for $Y_i = \mu_i + z_i$ with $z_i \sim \Ncl(0, 1)$. The goal is to test $H_{0, i} : \mu_i = 0$ against $H_{1, i} : \mu_i \not = 0$.

  1. Sparse testing: in this regime, at most one $H_{1, i} : \mu_i \not = 0$.
    • For $0 < \mu_i < \sqrt{2 \log n}$ and $\mu_j = 0, i \not = j$ then any test with T1 error less than $\alpha$ cannot have power greater than $1-\alpha$, which is the same performance as guessing $\mathsf{Ber}(\alpha)$.
    • For $\mu_i > \sqrt{2 \log n}$ and $\mu_j = 0, i \not = j$ then Bonferroni's test (with any T1 error bound $\alpha$) has power approaching $1$ as $n \to \infty$.
  2. Distributed testing: in this regime, many $H_{1, i}$ can be true, but we assume that $\|\mu\|_2^2$ is controlled. Define the signal-to-noise ratio $\theta_n = \|\mu\|^2 / \sqrt{2n}$.
    • For $n \gg 1$, an approximation with CLT implies that this testing problem is almost the same as the simple problem of distinguishing $H_0 : Z_n \sim \Ncl(0, 1)$ from $H_1 : Z_n \sim \Ncl(\theta_n, 1 + \frac{\theta_n}{\sqrt{n/8}})$.
    • If $\theta_n \to 0$, distinguishing the two hypotheses is impossible, whereas the test has nontrivial power as soon as $\theta_n > 0$ and it increases as $\theta_n \gg 1$.
    • Interestingly, the distributed testing problem fails to detect sparse alternatives of size $\sqrt{2 \log n} \ll \sqrt{2n}$.
