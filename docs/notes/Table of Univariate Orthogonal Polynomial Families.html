Table of Univariate Orthogonal Polynomial Families

There are a variety of orthogonal polynomial families for different weighted $\Lcl^2(\mu)$ spaces.

| Space | Family| Three-term Recurrence | | ------ | ------ | - | | $\mu = \Ncl(0, 1)$ | Hermite, $He_n$ | $He_{n+1} (x) = x He_{n}(x) - n He_{n-1}(x)$| | $\mu = \mathsf{Unif}(-1, 1)$ | Legendre, $P_n(x)$ | $(n+1) P_{n+1}(x) = (2n+1)xP_n(x) - n P_{n-1}(x)$ | | $\mu = \mathsf{Beta}(a, b)$ on $[0, 1]$ | Jacobi, $J_n^{a, b}(x)$ | | | $\mu = \Gamma(a, b)$ | Laguerre, $L_n^{a, b}(x)$ | |

When evaluating these polynomials, it is very important to use specialized numerically stable methods such as Clenshaw's Algorithm/Three Term Recurrence Relation.
