Vapnik-Chervonenkis Theory

Vapnik-Chervonenkis theory describes a set of methods for bounding the complexity of a set of sets $\Ccl \subseteq 2^{\Xcl}$, or more generally, the complexity of a function class $\Fcl$. A bound on the VC dimension of $\Ccl$ can be directly translated into a CLT-like rate of uniform convergence, $$\E_{X_1, \ldots, X_n}\left[\sup_{c \in \Ccl} |\mu_n(C) - \mu(C)|\right] \lesssim n^{-1/2}$$ and in this situation, $\Ccl$ is said to be a Glivenko-Cantelli class. Similarly, for a 'scale sensitive' notion of the VC dimension of $\Fcl$, bounds on its dimension can be translated directly into uniform convergence bounds for the associated empirical process $(G_n(f))_{f \in \Fcl}$.

VC Theory for Sets

Definition (VC Dimension). Let $\Xcl$ be a set and let $\Ccl \subseteq 2^\Xcl$ contain subsets of $\Xcl$. We say that a set $I \subseteq \Xcl$ is shattered by $\Ccl$ if $|\Ccl \cap I| = 2^{|I|}$, where $\Ccl \cap I \coloneqq \{ C \cap I : C \in \Ccl\}$. In other words, $I$ is shattered by $\Ccl$ if for every $J \subseteq I$, there exists $C \in \Ccl$ so that $C \cap I = J$.

The VC dimension $\mathsf{VC}(\Ccl)$ is maximal cardinality of a set shattered by $\Ccl$.

As a consequence of the Sauer-Shelah Theorem, the VC dimension of a set is a crude bound on the covering number of the set. In particular, we have that

Corollary (of the Sauer-Shelah Theorem). Let $\mu_n$ be a measure supported on $n$ points $\Xcl = \{X_1, \ldots, X_n\}$. Then $$ \log \Ncl(\Ccl, \|\cdot\|_{\Lcl^\infty(\mu_n)}, \eps) \lesssim \mathsf{VC}(\Ccl) \log(n) \qquad 0 < \eps < 1 $$ whereas for $\eps \geq 1$, $\Ncl(\Ccl, \|\cdot\|_{\Lcl^\infty(\mu_n)}, \eps) = 1$ trivially. It follows by symmetrization that for any measure $\mu$, and for its empirical approximation $\mu_n = \frac{1}{n}(\delta_{X_1} + \dots + \delta_{X_n})$ where $X_i \sim \mu$ i.i.d., we have that $$ \begin{aligned} \E\left[ \sup_{C \in \Ccl} |\mu_n(C) - \mu(C)|\right] & \lesssim \frac{1}{\sqrt{n}} \E\left[ \int_0^\infty \sqrt{\log \Ncl(\Ccl, \|\cdot\|_{\Lcl^2(\mu_n)}, \eps)} \right] \\ & \leq \frac{1}{\sqrt{n}} \E\left[ \int_0^1 \sqrt{\log \Ncl(\Ccl, \|\cdot\|_{\Lcl^\infty(\mu_n)}, \eps)} \right] \\\ & \leq \sqrt{\frac{\mathsf{VC}(\Ccl) \log n}{n}} \end{aligned} $$

It is in fact possible to remove the $\log n$ factor using the technique of probabilistic extraction (cf. Van Handel Section 9.1, Lemma 9.10), which leads to the following bound.

Theorem (Dudley). There is a universal constant $K$ such that $$ \sup_\mu \Ncl(\Ccl, \|\cdot\|_{\Lcl^2(\mu)}, \eps) \leq \left( \frac{K}{\eps}\right)^{K \, \mathsf{VC}(\Ccl)} $$

VC Theory for Function Classes

It is in fact possible to generalize much of the above logic to function classes $\Fcl$ that are not merely indicators of sets. First, we define a generalization of the VC dimension.

Definition (scale-sensitive VC dimension). Let $\Fcl$ be a function class from $\Xcl \to \R$. Let $I \subset \Xcl$ and $h : \Xcl \to \R$. The pair $(I, h)$ is $\eps$-shattered by $\Fcl$ if, for any $J \subseteq I$, there exists $f \in \Fcl$ such that $$ f(x) - h(x) \geq \eps, \quad x \in J \qquad \qquad f(x) - h(x) \leq 0,\quad x \in I \setminus J $$

In this setting, the Sauer-Shelah Theorem is replaced by the Mendelson-Vershynin Theorem, which implies that

Corollary (of Mendelson-Vershynin Theorem). Let $\Fcl$ be a uniformly bounded function class: $\sup_{f \in \Fcl} \|f\|_\infty \leq 1$. Let $\mu$ be a measure, and let $(G_n(f))_{f \in \Fcl}$ be the empirical process, $$ G_n(f) = \frac{1}{\sqrt{n}}\sum_{i=1}^n (f(X_i) - \E_\mu[f(X)]) \qquad X_i \sim \mu\, \text{ i.i.d.} $$ Then, $G_n$ is Lipschitz with constant $C \leq 1$, and its metric entropy satisfies $$ \log \Ncl(\Fcl, \|\cdot\|_{\Lcl^2(\mu)}, \eps) \lesssim \mathsf{VC}(\Fcl, \eps/K) \log(\eps^{-1}) $$ and so by Dudley's bound, $$ \E \left[ \sup_{f \in \Fcl} G_n(f)\right] \lesssim \inf_{\eps \geq 0} \left\{ \eps + \int_\eps^\infty \sqrt{\mathsf{VC}(\Fcl, \eps/K) \log(\eps^{-1})} \, d \eps\right\}. $$
