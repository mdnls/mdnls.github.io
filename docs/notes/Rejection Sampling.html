Rejection Sampling

A Monte-Carlo method for generating samples from an absolutely continuous distribution $\pi(y) \in \Pcl(\R)$. This method requires access to samples of a distribution $g(y) \in \Pcl(\R)$ which majorizes $\pi$: $$ \sup_{y \in \R} \frac{\pi(y)}{g(y)} \leq C $$

Algorithm:

  1. Sample $y_i \sim g$ and $u_i \sim [0, 1]$.
  2. If $u_i \leq \frac{\pi(y)}{Cg(y)}$, return $y_i$.
  3. If not, reject $y_i$ and repeat from Step 1.

Proof: this algorithm can be viewed as change-of-measure by the density ratio. Let $\phi(y)$ be a test function: $$ \begin{aligned} \int \phi(y)\left[ \int _0^1 1\left[u \leq \frac{\pi(y)}{Cg(y)}\right]\, du \right] \, g(y) dy = \frac{1}{C} \int \phi(y) \left(\frac{\pi(y)}{g(y)}\right) g(y) \, dy = \frac{1}{C} \int \phi(y) g(y) \, dy \end{aligned} $$ So, for $y_i$ generated by the above algorithm, $\phi(y_i)$ has the correct value on average.
