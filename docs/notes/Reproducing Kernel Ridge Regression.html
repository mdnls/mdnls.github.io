Reproducing Kernel Ridge Regression

Let $\{(x_i, y_i)\}_{i=1}^n$ be a sequence of points let $K(x, y)$ be the kernel of an RKHS $\Hcl$. Kernel Ridge Regression is the least-squares problem, $$ \min_{f \in \Hcl} \frac{1}{2n} \sum_{i=1}^n(y_i - f(x_i)) + \frac{\lambda}{n}\|f\|_{\Hcl}^2. $$ which can be thought of as "infinite dimensional regression" regularized by the RKHS norm.

Note: there are two types of kernels that are often discussed in similar contexts. An inner product kernel, which we discuss in this page, is a PSD bilinear function $K(x, y) \to \R$ that can be used to define 'inner product-like' quantities. It is also common to work with distance kernels, which are positive valued functions $K(x, y) \to \R_+$ that define distance-like quantities. See also:

  • Smoothing Estimators for Nonparametric Regression and Kernel Density Estimation, which uses a distance kernel to interpolate observed data for regression.
  • Reproducing Kernel Ridge Regression, which uses an inner product kernel to solve regularized regression in an RKHS.

(Proposition). The KRR solution $\hat{f}$ can be written in closed form as $$ \begin{aligned} \hat{f}(x) = \frac{1}{\sqrt{n}}\sum_{i=1}^n \hat{\alpha}_i \, K(x, x_i) \end{aligned} $$ with coefficients $\hat{\alpha} = (\hat{K} + \lambda I_n/n)^{-1} \frac{y}{\sqrt{n}}$, where $\hat{K}_{ij} = \frac{1}{n} K(x_i, x_j)$ is the kernel gram matrix.

Proof. first, we justify the so-called 'kernel trick,' which is that this seemingly infinite dimensional optimization problem over $f \in \Hcl$ can be reduced to a $n$-dimensional problem.

Consider the subspace $L\subseteq \Hcl$ given by, $$ L \coloneqq \mathsf{span}\left( K(\cdot, x_1), \ K(\cdot, x_2), \ \ldots, \ K(\cdot, x_n)\right) $$ Let $f^L = \mathsf{Proj}_L (f)$ and $f^\perp = f - f^L$. The fact that $f^\perp$ is orthogonal to $L$ is equivalent to, $$ f^\perp(x_i) = \langle f, K(\cdot, x_i) \rangle = 0 $$ and since $f^\perp$ has no effect on $\sum_{i=1}^n (f(x_i) - f^\perp(x_i) - f^L(x_i))^2$, the minimum solution has $f^\perp = 0$. We can also write $$ f^L(x) = \sum_{i=1}^n \alpha_i K(x, x_i) \implies \sum_{i=1}^n (f^L(x_i) - y_i)^2 = \|K\alpha - y\|_2^2 $$ and $$ \|f^L\|_{\Hcl}^2 = \left\langle \sum_{i=1}^n \alpha_i K(\cdot, x_i), \sum_{j=1}^n \alpha_j K(\cdot, x_j)\right \rangle = \sum_{i,j = 1} \alpha_i \alpha_j K(x_i, x_j). $$ so that, overall, the KRR problem is equivalent to $$ \min_{\alpha \in \R^n} \frac{1}{2n} \|y - K\alpha\|_2^2 + \frac{\lambda}{n} a^T K a $$ which is standard ridge regression.

Remark: at a high level, the idea behind the 'kernel trick' is that, due to the representer theorem, the pointwise condition $f(x_i) - y_i = 0$ is equivalent to the linear constraint $\langle f(\cdot) - y_i, K(\cdot, x_i) \rangle = 0$, which defines an $n$-dimensional space over which to optimize.
