Fisher's Combination Test

Definition. consider a multiple testing problem with nulls $H_{0, i}$ and alternatives $H_{1, i}$ for $i = 1 \ldots n$. Suppose that for each test, we observe a uniform p-value, ie. a random variable $p_i$ that is $\mathsf{Unif}([0, 1])$ under $H_{0, i}$. The Fisher Combination Test performs the following:

  1. Compute $Y_i = F^{-1}(p_i) : = - \log p_i$. The function $F^{-1}$ is the inverse CDF of $\mathsf{Exp}(1)$, so $Y_i \sim \mathsf{Exp}(1)$ under the null (cf. Inverse Transform Sampling).
  2. Compute the test statistic $T_n = \sum_{1 \leq i \leq n} Y_i$. It is a $\Gamma(1, n)$ random variable under the null.
  3. If $G(T_n) > 1 - \alpha$, reject the global null, where $G(x)$ is the CDF of $\Gamma(1, n)$. Otherwise, fail to reject.

Note: an alternative way to state this test is to use the fact that $2 Y_i \stackrel{d}{=} \chi^2_{\nu=1}$, a chi-squared distribution with one degree of freedom.
