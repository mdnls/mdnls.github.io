Gaussian Distribution

The Gaussian distribution on $\R$, denoted $\Ncl(\mu, \sigma^2)$, is a two parameter distribution with density $$ p(x) = \frac{1}{\sqrt{2 \pi \sigma^2}} e^{-x^2/2} \qquad \iff \qquad X \sim \Ncl(\mu, \sigma^2). $$ The parameter $\mu \in \R$ is the mean and $\sigma^2 \in \R_+$ is the variance. This distribution has fundamental importance in probability and statistics.

The multivariate Gaussian $\Ncl(\mu, \Sigma)$ is a generalized probability distribution over $\R^d$, with mean $\mu \in \R^d$ and covariance matrix $\Sigma \in \R^{d \times d}$, which must be a positive definite matrix. Its density is $$ p(x) = \frac{1}{(2 \pi)^{d/2} \sqrt{\det \Sigma}} \exp \left( (x - \mu)^T \Sigma^{-1} (x - \mu)\right). $$ This page serves as a reference for useful facts about Gaussian distributions.

Properties

Characteristic Function

If $X \sim \Ncl(\mu, \Sigma)$ then $$ \phi_X(t) \coloneqq \E[e^{itX}] = \exp\left(\langle t, \mu\rangle + \frac{1}{2}t^T \Sigma t\right). $$

Independence

If $X, Y$ are jointly multivariate Gaussian, then they are independent if and only if $\Cov(X, Y) \coloneqq \E[X Y^T] = 0$, a property which is not true in general.

  • For arbitrary independent random vectors $X, Y$, independence implies that any moment factorizes: $\E[|X_i|^\alpha |X_j| ^\beta] = \E[|X_i|^\alpha] \E[|X_j|^\beta]$ for all $\alpha, \beta \in \N$.
  • If this property holds (and if all these moments are finite) then by Taylor expansion, one can prove that the MGF factorizes: $\phi_{X, Y}(t, s) = \phi_X(t) \phi_Y(s)$ which implies independence.
  • The surprising property for Gaussians is that just $\alpha = \beta = 1$ is sufficient.

See also: Independent Random Variables

Proposition. If $X, Y$ are both marginally $\Ncl(0, 1)$ distributed, and if $\E[XY] = \E[X]\E[Y] = 0$, then they are independent.

Proof. the idea is that if $\E[XY] = 0$ then the covariance matrix $\Sigma = \Cov([X \quad Y])$ is an identity matrix. The characteristic function is $$ \phi_{X, Y}(t, s) = \exp \left( \frac{t^2}{2} + \frac{s^2}{t}\right) = \phi_{X}(t) \phi_Y(s) $$ which implies independence.

Note. one could imagine a higher order version of this proof. If there were a distribution with characteristic function $\phi_X(t) = \exp \left( \langle a, t \rangle + t^T B t + C[t^{\otimes 3}]\right)$, where $C$ is a third order moment tensor which is applied to $t^{\otimes 3}$, then it would factor whenever $B$ and $C$ are diagonal. But it doesn't seem like any such distributions exist with a nice closed form density...

Tail Decay

For the univariate Gaussian, its complementary CDF $\bar{F}(t) \coloneqq P[X > t]$ is in fact comparable to the normal density $p(x) \propto e^{-x^2/2}$. This is a prototypical example of a sub-Gaussianity. In particular, Mill's Ratio is the quantity $$ m(t) = \frac{\bar{F}(t)}{p(t)} $$ and it satisfies $$ \frac{t}{t^2+1} < m(t) <\frac{1}{t}, $$ a fact which is often useful for proving upper and lower tail bounds.
