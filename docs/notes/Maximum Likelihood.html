Maximum Likelihood

Maximum likelihood estimation is an extremely popular framework for designing and analyzing parameter estimation methods.

Definition: given a parametric family $\{p_\theta : \theta \in \Theta\}$ and samples $x_1, \ldots, x_n$, the maximum likelihood estimator $\th_n$ is the solution of, $$ \begin{aligned} \th_n = \arg \sup_{\theta \in \Theta} \frac{1}{n} \sum_{i=1}^n \log p_\theta(x_i). \end{aligned} $$ This is equivalent to (empirically) minimizing $\KL(\theta^* \mid \theta)$, $$ \begin{aligned} \KL(\theta^* \mid \theta ) = \E_{\theta^*}\left[ \log \frac{p_{\theta^*}(x)}{p_\theta(x) } \right] \approx -\frac{1}{n}\sum_{i=1}^n \log p_\theta(x_i) + C \end{aligned} $$

From the perspective of statistical analysis, there are a few key properties one would like the MLE to satisfy:

  • Identifiability: $p_{\theta_1} = p_{\theta_2} \implies \theta_1 = \theta_2$.
  • Strong identifiability: for all $\eps > 0$, $\KL(p_{\theta_1} \mid p_{\theta_2}) > 0$ for all $\|\theta_1 - \theta_2\| \geq \eps$.
  • Consistency: $\th_n^{MLE} \stackrel{p}{\longrightarrow} \theta^*$.

Theorem: the following conditions are sufficient for consistency of maximum likelihood:

  • Quasi-Lipschitz: $\mid \log p_\theta(y) - \log p_{\theta'} (y) \mid \leq L \phi(\|\theta - \theta'\|_2)$ for some $\phi : \R_+ \to \R_+$ such that $\lim_{t\to 0} \phi(t) = 0$.
  • Integrable envelope: there exists a function $\Psi(y) : \Ycl \to \R$ such that $\mid \log p_\theta(y)\mid\, \leq \Psi(y)$ and $\E_{\theta^*}\Psi(y) < \infty$.
  • Quasi-Lipschitz + Integrable envelope: $\mid \log p_\theta(y) - \log p_{\theta'} (y) \mid \leq \Psi(y) \phi(\|\theta - \theta'\|_2)$ where $\Psi, \phi$ satisfy the conditions above.

The proof of this theorem is discussed in more generality in M- and Z- Estimation
