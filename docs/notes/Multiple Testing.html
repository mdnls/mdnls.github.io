Multiple Testing

Description: multiple testing is a generalization of simple testing where there can be multiple null hypotheses. For example, in the Multiple Testing Gaussian Sequence Model, we consider a sequence of observations $Y_i \sim \Ncl(\mu_i, 1)$, and we test the set of hypotheses: $$ H_{0, i} : \mu_i = 0 \qquad H_{1, i} : \mu_i \not = 0. $$ for $i = 1 \ldots n$. The goal is to perform multiple simultaneous hypothesis tests, and to control for situations such as dependencies between the tests.

Performance Metrics:

  • Family Wise Error Rate: let $I \subseteq [n]$ and call $H_I$ the situation where $H_{0, i}$ holds for $i \in I$, and that $H_{1, i}$ holds for $i \not \in I$. The FWER is $P_{H_I}(\text{at least one } \{ H_{0, i} : i \in I\} \text{ is rejected})$. We say that a test 'controls the FWER at level $\alpha$' if this probability is bounded by $\alpha$ for all $I \subseteq n$.
  • Weak FWER: a test is said to 'weakly control FWER' if it satisfies the previous definition when $I = [n]$.
  • False Discovery Rate: a test controls FDR at rate $\alpha$ if, for all $I \subseteq [n]$,

$$ \E_{H_I}\left[\frac{\#\{i \in I : H_{0, i} \text{ is rejected}\}}{\#\{ i \in [n] : H_{0, i} \text{ is rejected}\}}\right] \leq \alpha. $$ The idea of this metric is that, given an output rejection set $R \subseteq [n]$, the fraction of indices in $R$ that were actually null (the false discoveries) is bounded by $\alpha$.

Note that, when $I = [n]$, quantity inside the expectation becomes $\{0, 1\}$-valued and in fact it coincides with the Weak FWER.

Remarks:

  • In simple testing and composite testing, we analyze tests like the Generalized Likelihood Ratio Test which are tailored to specific null and alternative distributions (ie. by using their log-likelihood functions). Instead, multiple testing abstracts away the specific form of each hypothesis, typically via an assumption like having access to a Super-uniform p-value. This way, we can focus on designing rejection procedures, which take p-values as input and which give rejection sets as output.
