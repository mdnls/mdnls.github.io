Stochastic Processes

  • Introduction: Stochastic Processes and Gaussian Process
    • Sampling Gaussian processes requires efficiently sampling a multivariate Gaussian. For example, using Cholesky or Eigenvalue decomposition.
  • Karhunen-Loeve Expansion
    • Looking forward to Reduced Order Modeling: The KL expansion allows to represent a random field using a finite number of coefficients which are random, but constant in space. Then, functionals $Y = F(X(\cdot, \omega))$ can be thought of approximately as $Y \approx f(Z_1(\omega), \ldots, Z_r(\omega))$ to which we can apply sensitivity analysis methods.
  • Wiener Chaos Expansion
    • approximation of a random variable $Y$ by a polynomial function of random variables $(X_1, \ldots, X_d)$.
    • A.K.A. Polynomial Chaos Expansion
    • "It is widely used in stochastic finite element analysis and as a surrogate model to facilitate uncertainty quantification analyses." (according to wikipedia)
